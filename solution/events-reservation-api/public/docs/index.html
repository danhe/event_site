    Introduction

    Welcome to the events-reservation API! You can use my API to access events-reservation API endpoints, which can get information on various events, users and invitations in my database.

    This example API documentation page was created with Slate.

    Sessions

    Create a new session(login)

    axios.post('https://events-reservation-api.herokuapp.com/api/v1/sessions', 
  {
    password: <PASSWORD>,
    email: <EMAIL>
  })
  .then((response) => {
  })
  .catch((e) => {
  })

    curl "https://events-reservation-api.herokuapp.com/api/v1/sessions"
  -X POST
  -H "Content-Type: application/json"
  -d '{"email":<EMAIL>, "password": <PASSWORD>}'

    The above command returns JSON structured like this:

    {
  created_at: "2019-07-15T17:30:35Z",
  email: "admin@events.com",
  id: 1,
  name: null,
  role: "admin",
  token: "DpibvwPkN7VWShdNTu5JX2WY",
  updated_at: "2019-07-15T17:30:35Z",
}

    This endpoint retrieves a new session(login), firstly we need to login to get a user's secure token.

    Once we get the token, we need to add the token in the request header to make sure the communication with API works.

    'Authorization': Bearer token="<token>", email="<email>"

    HTTP Request

    POST https://events-reservation-api.herokuapp.com/api/v1/sessions

    Errors

    This error section is stored in a separate file in includes/_errors.md. Slate allows you to optionally separate out your docs into many files...just save them to the includes folder and add them to the top of your index.md's frontmatter. Files are included in the order listed.

    The Kittn API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- The kitten requested is hidden for administrators only.
    404 Not Found -- The specified kitten could not be found.
    405 Method Not Allowed -- You tried to access a kitten with an invalid method.
    406 Not Acceptable -- You requested a format that isn't json.
    410 Gone -- The kitten requested has been removed from our servers.
    418 I'm a teapot.
    429 Too Many Requests -- You're requesting too many kittens! Slow down!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
